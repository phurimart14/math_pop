# Math Pop AI

เกมบวกเลขหรรษาท้าทายเวลา ที่สร้างด้วย HTML, CSS (Tailwind) และ JavaScript ทั้งหมดอยู่ในไฟล์เดียว `index.html`

## คอนเซ็ปต์

Math Pop AI เป็นเกมฝึกคิดเลขสำหรับเด็ก โดยใช้กล้องจับการเคลื่อนไหวของนิ้วชี้ผ่าน MediaPipe Hands ถ้าไม่ใช้กล้องก็สามารถเล่นด้วยเมาส์หรือทัชสกรีนได้

## คุณสมบัติหลัก

- ป๊อปฟองคำตอบด้วยนิ้วชี้หรือเมาส์
- สร้างโจทย์เลขบวกแบบสุ่ม (ผลรวมไม่เกิน 10)
- เก็บคะแนนและสถิติสูงสุดไว้ใน `localStorage`
- มีโหมดเวลา 30 วินาที และ 60 วินาที
- ใช้เสียงประกอบด้วย Web Audio API
- เลย์เอาต์สวยงาม รองรับมือถือและเดสก์ท็อป
- มี leaderboard ภายในเครื่องสำหรับ 5 คะแนนสูงสุด

## ไฟล์ในโปรเจกต์

- `index.html` 
  - โค้ดทั้งหมดของเกม
  - ส่วน UI, สไตล์ และสคริปต์ JavaScript อยู่รวมกัน

## วิธีเล่น

1. เปิด `index.html` ในเว็บเบราว์เซอร์ (แนะนำ Chrome หรือ Edge)
2. ใส่ชื่อผู้เล่น
3. เลือก 30 วินาที หรือ 60 วินาที
4. กดปุ่ม "เปิดกล้อง & เริ่มท้าทาย!" หากต้องการเล่นโดยไม่ใช้กล้อง ให้กด "เล่นแบบใช้เมาส์คลิกแทน"
5. ใช้นิ้วชี้หรือเมาส์ลากไปที่ฟองคำตอบที่คิดว่าเป็นผลลัพธ์ถูกต้อง
6. เมื่อเวลาหมด ระบบจะแสดงคะแนนและบันทึก leaderboard ในเครื่อง

## การตั้งค่าและพัฒนา

- โครงการนี้ไม่ต้องติดตั้งอะไรเพิ่ม
- เปิดไฟล์ `index.html` ด้วยเบราว์เซอร์ทันที
- หากต้องการพัฒนาเพิ่มเติม สามารถแก้ไขไฟล์ `index.html` ได้โดยตรง

## หมายเหตุ

- ฟีเจอร์กล้องใช้ MediaPipe Hands และอาจต้องให้สิทธิ์การเข้าถึงกล้องของเว็บเบราว์เซอร์
- หากกล้องไม่สามารถใช้งานได้ จะเล่นด้วยเมาส์/ทัชได้ตามปกติ
